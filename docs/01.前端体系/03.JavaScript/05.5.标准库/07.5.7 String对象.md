### 1.概述
`String` 对象是 JavaScript 原生提供的三个包装对象之一，用来生成字符串对象。
```javascript
var s1 = 'abc';
var s2 = new String('abc'); //字符串对象：类似数组的对象

typeof s1 // "string"
typeof s2 // "object"

s2.valueOf() // "abc"
```
 除了用作构造函数，String对象还可以当作工具方法使用，将任意类型的值转为字符串。
```javascript
String(true) // "true"
String(5) // "5"
```
